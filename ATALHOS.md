# Atalhos de Teclado — Araujo Prev

## Navegação

| Atalho     | Ação                        |
|------------|-----------------------------|
| Alt + G    | Ir para Gerar Recibo        |
| Alt + H    | Ir para Histórico           |
| Alt + C    | Ir para Clientes            |
| Alt + A    | Ir para Administrativo      |

## Formulário de Recibo

| Atalho     | Ação                                          |
|------------|-----------------------------------------------|
| Enter      | Gerar recibo (quando campos preenchidos)      |
| Alt + L    | Limpar todos os campos                        |
| Escape     | Fechar janela de detalhes                     |

## Data

| Ação                          | Comportamento                              |
|-------------------------------|--------------------------------------------|
| Clicar no campo Dia           | Preenche automaticamente com a data de hoje|
| Digitar o dia (2 dígitos)     | Avança automaticamente para o Mês         |
| Selecionar o Mês              | Avança automaticamente para o Ano         |

## Backup e Restauração

| Ação        | Como fazer                                          |
|-------------|-----------------------------------------------------|
| Backup      | Menu lateral → Backup (salva em Documentos)         |
| Restaurar   | Menu lateral → Restaurar → selecionar arquivo .json |
